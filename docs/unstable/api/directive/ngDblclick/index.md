<h1 class="api-title">

  ng-dblclick



</h1>





This wrapped on `ngDblclick` directive allows you to specify custom behavior on a dblclick event on a fa-surface .






  
<h2 id="usage">Usage</h2>
  
```html
<ANY ng-dblclick="expression">

</ANY>
```
